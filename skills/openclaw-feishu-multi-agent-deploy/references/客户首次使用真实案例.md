## 如果客户只需要 1 个群

最简单的做法：

- 删除 `external_main`
- 只保留一个 `team`
- 保留该群真正需要的角色

例如：

- 只需要 `supervisor + ops`，就删除财务 worker
- 同时把 `workflow.stages` 改成只包含 `ops`

## 替换说明

把下面这些真实值替换成客户自己的值：

- `accounts[].appId`
- `accounts[].appSecret`
- `accounts[].encryptKey`
- `accounts[].verificationToken`
- `teams[].group.peerId`
- `teams[].displayName`
- `systemPrompt`
- `name / description / role / responsibility`

## 推荐使用方式

1. 复制这份案例
2. 替换成客户真实值
3. 保存成客户自己的输入文件
4. 再配合 `客户首次使用-Codex提示词.md` 一起交给 Codex
